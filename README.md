# Alcohol-consumption-data-analysis-using-Python-programming
Analyzed a real-world dataset using python programing knowledge. Data was cleaned and standardized, then sorted and extracted the top 15 countries with total alcohol consumption and then visualized the insights using a horizontal bar chart in Matplotlib.


What i did 
- Cleaned and standardized dataset columns
- Checked for missing values from the columns
- sorted countries by total litres of pure alcohol consumed per country
- Selected the top 10 countries
- Created a horizontal bar chart to visualized the total consumption in these countries

The tools i used 
- Python
- pandas
- Matplotlib
